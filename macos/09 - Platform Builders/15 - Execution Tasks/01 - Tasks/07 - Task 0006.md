# Task 0006 — Real Payload Execution

## Objective
Execute a real payload through the full ICOS pipeline.

## Inputs
- real user request payload
- routing and policy configuration

## Execution Steps
1. submit payload to Gateway Core
2. validate routing via Request Router
3. resolve via Execution Routing
4. execute through Execution Kernel
5. process across Runtime Modules
6. invoke Model Interface (selected provider)
7. enforce Sandbox Isolation
8. stream via Stream Handler
9. normalize via Response Engine

## Expected Output
- correct, normalized response
- deterministic path adherence
- no errors across pipeline

# Native Builder

## Role
System responsible for deterministic generation of native applications within ICOS.

## Core Function
- generates Swift/iOS project structures
- enforces modular architecture
- integrates with execution pipeline
- prepares local build environments

## Capabilities
- Swift project scaffolding
- iOS application structure generation
- macOS application templates
- local development environment setup

## Constraints
- must follow modular architecture rules
- no monolithic structure generation
- no deviation from platform standards

## Position in Flow
Execution Kernel → Native Builder → Build System
